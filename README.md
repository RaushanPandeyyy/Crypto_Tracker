🚀 Live Cryptocurrency Tracker

A responsive cryptocurrency tracking web application built using HTML, CSS, and JavaScript.

📌 Features

- Real-time cryptocurrency prices
- Coin rankings
- Coin symbols
- 24-hour price change tracking
- Search functionality
- Auto-refresh every 60 seconds
- Responsive design for mobile and desktop
- Data fetched using CoinGecko API

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- CoinGecko API

📊 Project Functionality

This application fetches live cryptocurrency market data from the CoinGecko API and displays the top cryptocurrencies along with their current prices, rankings, symbols, and 24-hour market performance.

Users can search for specific cryptocurrencies and refresh the data to view the latest market information.

🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open "index.html" in your browser.

No installation or additional dependencies are required.

📸 Features Included

- Dynamic API Integration
- DOM Manipulation
- Async/Await
- Fetch API
- Search Filter
- Responsive User Interface

🔗 API Used

CoinGecko Public API
